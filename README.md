# 🚀 AI Future Skills Predictor

An AI-powered web application that predicts future demand for technologies using Machine Learning.

## 🔍 Features
- Predict future demand of tech skills
- Interactive charts and visualization
- Clean dark-themed UI
- Top 3 future technologies ranking

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## 📊 How it works
- Trained Linear Regression models on past skill demand data
- Predicts future trends based on selected year
- Displays results using charts and metrics
